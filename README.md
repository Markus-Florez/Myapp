# Mecatronica-y-v 
gestión de diagramas eléctricos/mecánicos 
Lenguaje de programacion: Kotlin 
herramienta: android studio
base de datos: firebase
## Instrucciones de Instalación
Para instalar la aplicación en un dispositivo Android:

1. Descargar el APK: Acceda al sitio web oficial de distribución (ej. https://mycar.adsosena.site/ ) desde el navegador de su móvil y presione el botón "Descargar APK".
2. Permisos de Seguridad: Si es la primera vez que instala una app fuera de Google Play Store, Android solicitará permiso para "Instalar aplicaciones de fuentes desconocidas" para el navegador que esté usando. Acepte este permiso temporalmente.
3. Instalación: Una vez finalizada la descarga, abra el archivo MecatronicaYV.apk desde las notificaciones o su gestor de archivos y confirme la instalación.
4. Registro y Activación:
   - Abra la aplicación por primera vez.
   - Complete el registro utilizando su correo electrónico o su cuenta de Google.
   - Importante: Al finalizar el registro, verá un mensaje indicando que su cuenta está "Pendiente". Un administrador del sistema recibirá la notificación y deberá aprobar su solicitud manualmente antes de que pueda acceder al contenido.
